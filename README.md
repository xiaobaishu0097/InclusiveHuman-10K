**InclusiveHuman-10K (IH-10K)** is, to the best of our knowledge, the first large-scale human body
parsing dataset centered on individuals with limb deficiencies. It provides pixel-wise annotations for
**residual limbs, prosthetic components, and mobility assistive devices**, filling a long-standing gap in
human parsing benchmarks and enabling inclusive evaluation of an underrepresented population.

| | |
|---|---|
| **Images** | 10,555 |
| **Person instances** | 14,939 |
| **Segmentation masks** | 160,133 |
| **Part classes** | 61 (27 common + 34 disability-related) |
| **Scenes** | everyday, sports, clinical |
| **License** | CC BY-NC-SA 4.0 |

## Dataset access

The dataset is released for **non-commercial research only** and access requires an approved request,
in line with our institutional Human Research Ethics approval and the licenses of the source images.

1. **Email the authors** at [heming.du@uq.edu.au](mailto:heming.du@uq.edu.au) with your name,
   affiliation, and intended (non-commercial) research use.
2. Once approved, you will be granted access to the dataset on
   [Google Drive](https://drive.google.com/drive/folders/1kpdpEo1WG7CgheHFZFPR9Dv1-MI6nkwj?usp=sharing).
3. By using the data you agree to the CC BY-NC-SA 4.0 license.

> Opening the Drive link before approval will only show a request-access prompt — please email first.
> IH-10K **must not** be used for biometric identification (face recognition / re-identification) or
> for medical diagnosis or clinical decision making.
